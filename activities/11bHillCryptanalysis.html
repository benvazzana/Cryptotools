How it works

If we have a "crib" for a ciphertext that has been encrypted with the Hill Cipher, we can work out a list of possible keys.

Suppose we know that

here

is the first word in the ciphertext:

RWBA KEQS RMJ VKFOO.

We eneter "here" and "RWBA" into the Hill Cryptanalysis Tool and get the result:

Possible first rows are

1, 2
14, 2

Possible second rows are:

3, 5
16, 5

There are four possible matrices we can form. First we consider the matrix \(\begin{bmatrix}1 &2\\3&5\end{bmatrix}\). We use the Mod 26 Inverse of Matrix tool to find its inverse and get:

\(\begin{bmatrix}21 &2\\3&25\end{bmatrix}\)

Now we try to decrpt our original ciphertext.

HERE GEBF NOTH IAGD

We see the first four letters decode as expected, but the rest is gibberish.

Next we try \(\begin{bmatrix}1 &2\\16&5\end{bmatrix}\). We find its inverse is \(\begin{bmatrix}21 &2\\16&25\end{bmatrix}\). We again try to decrypt, but using this key we get:

HERE GOES NOTH INGQ

We have succefully decipherd the message" "Here goes nothing."
