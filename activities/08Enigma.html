The Enigma Machine

For this activity, we will use an Enigma simulator from another site. You may use either:

  • 101 Computing (This one is very pretty.) To access the settings, click on one of the rotor wheels. Be sure to click Apply Settings when you are done.
  • Daniel Palloks (Doesn't have the look of the actual machine, but a very clean website, and allows you to paste in the messages.)

Set up your Enigma machine as follows:

  • Enigma: M3
  • Reflector (Umkehrwalze): B
  • Wheel order (Walzenlage): III, I, II
  • Ring Settings (Ringstellung): A A A (1, 1, 1)

    [On the Palloks simulator, click on the Rings button to access these settings.]

  • Indicator Settings (Grundstellung): D E P

    [On the 101 Computing simulator, these are called the Initial Positions.]

  • Plugboard (Steckerbrett): AW, EH, FR, GO, VY

    • [On the Palloks simulator, for each pair, enter the two letters into a box. When you are done, click Activate.]

Exercises:
  1. Encipher the message, “Did this work?” with message key QWE. Set up the Enigma as above. Type QWE twice. You should get:
    KFR CQR
    The indicator settings should have moved from DEP to DEV. Now reset the indicator settings to our message setting QWE. Now type our message. You should get:
    ESYL SANC YSX
  2. Let’s decrypt the message:
    EKX KJT AJMX LVUA
    The first six letters represent the message setting. Reset the Enigma to the original settings above (with DEP as the indicator settings), type these six letters. They should give you a 3 letter indicator setting followed by a repetition of that setting. Now switch your Enigma to those indicator settings and type the rest of the message.

Group Message Exchange

It's super easy to mess up a setting on the Enigma machine, so please double check your work before posting it.

  1. Set up your Enigma with the setting at the top of the page.
  2. Pick a three letter message setting and type it into your Enigma twice.
  3. Reset the Enigma Indicator Settings to the three letters you chose.
  4. Type a message.
  5. In the Enigma Messages forum of the Student Messages section on Blackboard, post the six letters you got from typing your indicator setting twice followed by the cipher text of your message.
  6. Decrypt the messages from the other groups.
